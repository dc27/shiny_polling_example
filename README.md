# Shiny Polling Example

Demonstration of polling to update dashboard automatically given changes in
underlying source data.

## Usage

1. Clone this repo
1. Open `pollr.Rproj` in RStudio
1. Open `app.R`
1. Run App

## Requirements

|package | version |
|---|---|
|shiny| 1.6.0|
|shinydashboard|0.7.1|
|tidyverse|1.3.1|
